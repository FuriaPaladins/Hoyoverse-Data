# Hoyoverse Data
This repository includes semi auto-updating data scraped from the [genshin wiki](https://genshin-impact.fandom.com/wiki/Wish/History#Wishes_by_Type)  
Data is also gathered from [ambr.top](https://ambr.top/) to sort Characters and Weapons between 4* and 5* for the json file.  
Currently, includes banner data and banner assets.  

This project is maintained primarily for personal use for my Discord Bot, [Itto](https://bit.ly/itto_bot)  


## Usage - Genshin
You should be requesting the json [from this page](https://raw.githubusercontent.com/FuriaPaladins/Hoyoverse-Data/master/genshin_banners.json)  
In this repo, is a file called `genshin_dataclasses.py` which includes a dataclass wrapper for the json data for easy access.
